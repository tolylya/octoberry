# Test application

##How to install app

1. **Clone the project**. `git clone https://github.com/tolylya/octoberry.git`.
2. **Run in dev mode**. `npm run start`.

## Commands

1. Run in development mode: `npm run start`
2. Build production: `npm run build`
3. Run tests: `npm run test`
